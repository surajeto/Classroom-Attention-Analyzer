# 📡 Classroom Attention Analyzer (ระบบวิเคราะห์ความตั้งใจในห้องเรียน)

ระบบวิเคราะห์ความสนใจและพฤติกรรมของนักเรียนในชั้นเรียนแบบเรียลไทม์โดยใช้ปัญญาประดิษฐ์ (AI) ในการตรวจจับผ่านกล้อง Webcam/USB ตัวระบบทำงานบนเบราว์เซอร์ผ่านอินเตอร์เฟสแบบพรีเมียม (Glassmorphism UI) รวดเร็ว ใช้งานง่าย และให้รายงานสรุปผลลัพธ์ที่เป็นประโยชน์ทันที

---

## ✨ ฟีเจอร์เด่นของระบบ (Key Features)

1. **AI Multi-Student Tracking & Pose Estimation**:
   - ติดตามใบหน้าและพฤติกรรมของนักเรียนหลายคนพร้อมกันโดยแยก ID อย่างชัดเจน ผ่านโมเดล **YOLOv8 Pose & Object Detection**
2. **การวิเคราะห์พฤติกรรมความสนใจ (Behavior Detection)**:
   - **ตั้งใจฟัง**: เมื่อดวงตาหันตรงมายังกล้อง
   - **มองไปทางอื่น**: เมื่อก้มหน้าหรือกรอกตามองไปทิศอื่น (Gaze & Gaze deviation estimation)
   - **เล่นมือถือ/มองคอมพิวเตอร์**: ตรวจหาวัตถุโทรศัพท์ (Phone) หรือโน้ตบุ๊ก (Laptop) ที่ทับซ้อนกับพิกัดตัวนักเรียน
   - **ลุกออกจากห้อง**: ตรวจสอบและนับครั้งสะสมเมื่อ ID ของนักเรียนหายไปจากจอนานเกินกว่าเวลาที่กำหนด
   - **นักเรียนคุยกัน**: ตรวจสอบระยะห่างหัวและใบหน้าระหว่างนักเรียนที่อยู่ติดกัน
3. **ระบบการจัดการกล้องระดับพรีเมียม (Advanced Camera Settings)**:
   - ตรวจสอบและแสกนกล้องที่เชื่อมต่ออัตโนมัติ (Camera Auto-Scan)
   - **ปุ่มเลือกพอร์ตโดยตรง (Camera Port Buttons)**: คลิกเลือกพอร์ตเพื่อเชื่อมต่อกล้องได้ทันที (ไม่ต้องมี Dropdown)
   - มีไฟสถานะการเชื่อมต่อกะพริบแสดงผล (Pulse Active Indicator)
   - ปุ่มสลับกล้องด่วน (🔀 Quick Switch) รองรับการวนเปลี่ยนกล้องทุกตัวที่แสกนพบ
4. **การบันทึกเวลาและส่งรายงานสรุปผล (Reporting & Export)**:
   - ตัวจับเวลาบันทึกกิจกรรม (Session Timer)
   - สรุปคะแนนความสนใจเฉลี่ยตลอดชั่วโมงเรียน (Overall Attention Score)
   - ส่งออกข้อมูลเป็น **Excel (.xlsx)** หรือรายงานสถิติเป็น **PDF (.pdf)**
   - บริการส่งรายงานเข้าสู่อีเมลปลายทาง (SMTP Email Service) ผ่านระบบ Gmail App Password

## 🛠 ความต้องการของระบบ (System Requirements)

- **ระบบปฏิบัติการ**: Windows, macOS หรือ Linux
- **Python**: เวอร์ชัน 3.10 ขึ้นไป (แนะนำ Python 3.11)
- **ฮาร์ดแวร์**: กล้อง Webcam ในตัว หรือกล้อง USB Webcam ภายนอก
- **พื้นที่เก็บข้อมูล**: อย่างน้อย 100 MB (สำหรับไลบรารีและโมเดล AI)

## 🚀 ขั้นตอนการติดตั้งและรันใช้งาน (Setup Guide)

### 1. ดาวน์โหลดโปรเจกต์
ดาวน์โหลดไฟล์ซอร์สโค้ดจากคลังข้อมูลนี้ หรือใช้ Git Clone:
```bash
git clone https://github.com/YOUR_USERNAME/classroom-attention-analyzer.git
cd classroom-attention-analyzer
```

### 2. ติดตั้งไลบรารีที่จำเป็น
รันคำสั่งด้านล่างเพื่อติดตั้งโมดูลผ่านไฟล์ `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 3. ดาวน์โหลดโมเดล YOLOv8 และ YOLOv8-Pose (จำเป็น)
ดาวน์โหลดไฟล์โมเดลสำเร็จรูปจากผู้พัฒนาอย่างเป็นทางการของ Ultralytics และนำมาวางไว้ที่โฟลเดอร์หลักของโปรเจกต์ (โฟลเดอร์เดียวกับ `main.py`):
1. **โมเดล YOLOv8 สำหรับการตรวจวัตถุ (โทรศัพท์/คอมพิวเตอร์)**:
   - ดาวน์โหลด [yolov8n.pt (คลิกดาวน์โหลด)](https://github.com/ultralytics/assets/releases/download/v8.2.0/yolov8n.pt)
2. **โมเดล YOLOv8 สำหรับการตรวจท่าทาง/จุดสำคัญ (ใบหน้า/ดวงตา)**:
   - ดาวน์โหลด [yolov8n-pose.pt (คลิกดาวน์โหลด)](https://github.com/ultralytics/assets/releases/download/v8.2.0/yolov8n-pose.pt)

### 4. การจัดการไฟล์โลโก้และพื้นหลัง
ตรวจสอบว่าในโฟลเดอร์ `image/` หรือ `static/image/` ของคุณมีไฟล์รูปภาพที่ระบบต้องการ:
- `images.jfif` (ใช้เป็นโลโก้ของระบบมุมขวาบน)
- `Gemin.png` (ใช้เป็นพื้นหลังแอปพลิเคชันหรือกรอบกล้องสำรอง)

### 5. เริ่มรันระบบ
เปิด Terminal ในโฟลเดอร์โปรเจกต์แล้วรันคำสั่งสตาร์ทเว็บเซิร์ฟเวอร์:
```bash
python main.py
```

เมื่อระบบเปิดสำเร็จ ให้เปิดเว็บเบราว์เซอร์ไปที่ลิงก์:
👉 **[http://localhost:8000](http://localhost:8000)**

## 📧 วิธีส่งรายงานทางอีเมล (SMTP Gmail setup)
ในการส่งรายงานสรุปสถิติผ่านหน้าเว็บโดยตรงไปยังอีเมลของอาจารย์หรือผู้บริหาร จำเป็นต้องเปิดการใช้งาน Gmail App Password:
1. เปิดการลงชื่อเข้าใช้แบบ 2 ขั้นตอน (2-Step Verification) ในหน้าบัญชี Google
2. ค้นหาคำว่า **"App Passwords"** หรือเข้าไปที่หน้าความปลอดภัยบัญชี
3. เลือกสร้างรหัสผ่านแอปใหม่ ตั้งชื่อแอปว่า "Classroom Analyzer" และจดรหัสผ่าน 16 ตัวอักษรไว้
4. บนหน้าเว็บ ให้เปิดหน้าต่างส่งอีเมล กรอกอีเมลผู้รับ อีเมลผู้ส่ง และกรอกรหัสผ่าน 16 หลักดังกล่าว เพื่อทำการส่งรายงานทันที
